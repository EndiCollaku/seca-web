# Seca

Seca is a modern, responsive web application built with **React**, designed to provide a seamless and interactive experience for coffee lovers and enthusiasts. The project focuses on clean design, smooth navigation, and dynamic user interactions.  

## Features
- Fully built with **React** for a fast and responsive UI.
- Hosted on **Vercel** for reliable performance and effortless deployments.
- Custom domain purchased via **Namecheap** for a professional web presence.
- Mobile-friendly and fully responsive design.

## Tech Stack
- **Frontend:** React, HTML, CSS, JavaScript
- **Hosting:** Vercel
- **Domain:** Namecheap
